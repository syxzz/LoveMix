# LoveMix 使用指南

## 快速开始

### 1. 首次使用

启动应用后,你会看到欢迎引导页,展示了 LoveMix 的四大核心功能:
- ✨ AI头像融合
- 🎨 纪念日卡片
- 🌟 虚拟约会
- 😊 表情包生成

点击"立即开始"进行注册,或点击"已有账号?立即登录"进行登录。

### 2. 注册账号

填写以下信息完成注册:
- 邮箱地址
- 用户名
- 密码(至少6位)
- 确认密码

注册成功后会自动登录并进入主页。

### 3. 主页功能

登录后进入主页,可以看到:
- 顶部显示你的昵称和爱心值
- 四个功能卡片,点击即可使用对应功能

## 核心功能

### AI头像融合 💑

1. 点击"AI头像融合"卡片
2. 上传两张照片(照片1和照片2)
3. 选择融合模式:
   - 未来宝宝: 生成你们未来宝宝的样子
   - 情侣头像: 生成融合后的情侣头像
4. 点击"AI生成"按钮
5. 等待生成完成,可以保存或分享结果

### 纪念日卡片 🎂

1. 点击"纪念日卡片"卡片
2. 填写纪念日信息:
   - 纪念日名称(如"恋爱一周年")
   - 选择日期
   - 双方昵称
3. 选择祝福风格:
   - 浪漫
   - 幽默
   - 文艺
4. 点击"生成祝福卡片"
5. 查看生成的祝福文案和卡片图片

### 虚拟约会 🌅

1. 点击"虚拟约会"卡片
2. 上传你们的照片
3. 选择约会场景:
   - 海边日落 🌅
   - 浪漫咖啡馆 ☕
   - 星空露营 ⭐
   - 未来之家 🏠
   - 樱花树下 🌸
   - 沙滩漫步 🏖️
4. 选择画面风格(写实/动漫/水彩/油画)
5. 点击生成,创造专属约会场景

### 表情包工坊 😊

1. 点击"表情包工坊"卡片
2. 上传照片
3. 选择表情包风格:
   - 可爱
   - 搞笑
   - 宠物
4. 输入想说的话
5. 生成专属表情包

## 个人中心

### 个人资料

在设置页面点击"个人资料"可以:
- 更换头像
- 修改用户名
- 设置手机号
- 查看会员等级
- 查看爱心值
- 查看使用统计

### 我的作品

查看所有生成的作品历史:
- 按类型筛选(全部/头像融合/卡片/约会/表情包)
- 查看作品详情
- 查看浏览量和点赞数

### 作品广场

浏览社区中其他用户分享的作品:
- 浏览精选作品
- 为喜欢的作品点赞
- 查看作品详情
- 升级为高级会员后可发布自己的作品

## 设置

### API密钥配置

如果你有自己的API密钥,可以配置:
- Replicate API Key: 用于AI图像生成
- OpenAI API Key: 用于文案生成

未配置时应用会使用模拟数据。

### 退出登录

在设置页面底部点击"退出登录"可以安全退出账号。

## 常见问题

### Q: 生成失败怎么办?
A: 请检查网络连接,或稍后重试。如果持续失败,请联系客服。

### Q: 如何获得更多爱心值?
A: 每日登录、完成任务、升级会员都可以获得爱心值。

### Q: 作品可以删除吗?
A: 在"我的作品"页面长按作品可以删除。

### Q: 如何分享作品?
A: 生成作品后点击分享按钮,可以分享到微信、朋友圈等平台。

### Q: 会员到期后会怎样?
A: 会员到期后会自动降级为免费版,之前生成的作品不会丢失。

### Q: 如何联系客服?
A: VIP会员可以在设置页面找到专属客服入口。

## 隐私与安全

- 所有数据存储在本地,保护你的隐私
- 密码经过加密处理
- 不会未经允许分享你的作品
- 可以随时删除账号和数据
