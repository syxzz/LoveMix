# 🎨 WelcomeScreen 现代化设计说明

## ✨ 苹果风格 + 液态玻璃效果

### 设计理念
参照 iOS 17 和 macOS Sonoma 的设计语言，采用液态玻璃（Glassmorphism）效果，营造现代、高级的视觉体验。

## 🎬 动画效果

### 1. 浮动气泡动画
- **8 个大小不一的气泡**：60-160px 随机大小
- **从底部向上浮动**：10秒完成一个循环
- **渐隐渐现**：透明度 0 → 0.4 → 0
- **缩放动画**：从 0 放大到 1，再缩小到 0
- **左右漂移**：随机 ±100px 水平移动
- **错开延迟**：每个气泡延迟 1 秒启动

### 2. Logo 动画
- **呼吸效果**：缩放 1.0 → 1.1 → 1.0（2秒循环）
- **轻微摆动**：旋转 -5° → 5° → 0°（9秒循环）
- **渐变光环**：半透明白色渐变背景
- **3D 阴影**：深度阴影营造立体感

### 3. 入场动画
- **淡入**：整体透明度 0 → 1（1秒）
- **滑入**：从下方 100px 滑入（弹簧动画）
- **缩放**：从 0.8 倍放大到 1.0 倍（弹簧动画）
- **弹簧参数**：tension: 40, friction: 8

## 🚀 性能优化

### 1. 原生驱动
- 所有动画使用 `useNativeDriver: true`
- 60fps 流畅运行

### 2. 气泡优化
- 8 个气泡，平衡视觉效果和性能
- 随机大小和位置，避免重复计算

### 3. 避免重渲染
- 使用 `useRef` 存储动画值
- 优化组件结构
