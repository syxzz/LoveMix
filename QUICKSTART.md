# LoveMix 快速开始指南

## 🚀 立即运行

```bash
# 1. 进入项目目录
cd LoveMix

# 2. 启动开发服务器
npx expo start

# 3. 选择运行平台
# - 按 'i' 在 iOS 模拟器运行
# - 按 'a' 在 Android 模拟器运行
# - 扫描二维码在真机运行
```

## 📱 首次运行体验

应用启动后，你会看到：

1. **首页** - 显示四个功能卡片和爱心值
2. 点击任意功能卡片进入对应页面
3. 所有功能都可以在**不配置API密钥**的情况下体验（使用模拟数据）

## 🎯 功能测试流程

### AI头像融合
1. 点击"AI头像融合"卡片
2. 上传两张照片（可以从相册选择或拍照）
3. 选择模式：未来宝宝 或 情侣头像
4. 点击"AI生成"按钮
5. 等待3秒，查看生成结果
6. 点击"保存"将图片保存到相册

### 纪念日卡片
1. 点击"纪念日卡片"卡片
2. 填写纪念日名称（如：恋爱一周年）
3. 选择日期
4. 输入双方昵称
5. 选择祝福风格
6. 点击"生成祝福卡片"
7. 查看生成的文案和卡片

### 虚拟约会
1. 点击"虚拟约会"卡片
2. 上传两张照片
3. 横向滑动选择约会场景
4. 选择画面风格
5. 点击"生成约会照"
6. 查看虚拟约会场景

### 表情包工坊
1. 点击"表情包工坊"卡片
2. 输入文本或点击快捷标签
3. 选择表情包风格
4. 点击"生成表情包"
5. 查看生成的4个表情包
6. 点击每个表情包的下载按钮保存

## ⚙️ 配置真实API（可选）

如果想使用真实的AI生成功能：

1. 从首页点击右上角设置图标（或创建设置入口）
2. 输入 Replicate API Key
3. 输入 OpenAI API Key（可选）
4. 点击"保存设置"

### 获取 Replicate API Key
- 访问：https://replicate.com
- 注册账户并登录
- 进入 Account Settings → API Tokens
- 创建新的 Token 并复制

### 获取 OpenAI API Key
- 访问：https://platform.openai.com
- 注册账户并登录
- 进入 API Keys 页面
- 创建新的 Secret Key 并复制

## 🐛 常见问题解决

### 问题1：模拟器无法启动
```bash
# iOS
sudo xcode-select --switch /Applications/Xcode.app

# Android
# 确保 Android Studio 已安装并配置好 AVD
```

### 问题2：依赖安装失败
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

### 问题3：图片选择器不工作
- iOS: 确保在模拟器中授予相册权限
- Android: 确保在设置中授予存储权限

### 问题4：TypeScript 错误
```bash
# 重启 TypeScript 服务器
# 在 VSCode 中按 Cmd+Shift+P，输入 "TypeScript: Restart TS Server"
```
